📖 YotonBase 使用指南

一、创建数据表

在左侧面板输入表名（建议英文）和列名（逗号分隔），点击"创建新表"。第一列自动设为主键，主键值不可重复。

示例：表名 students，列名 学号,姓名,年龄,专业

二、管理数据记录

  • 新增记录：选中表 → 点击"＋ 新增记录" → 填写表单 → 确认。主键字段必填。
  • 编辑记录：在表格行右侧点击"编辑"按钮，修改后可更新。主键不可修改。
  • 删除记录：点击行右侧"删除"按钮，确认后删除。该操作可撤销。

三、搜索与筛选

在工具栏列名输入框中填入要搜索的字段名，在搜索值中填入匹配内容，点击"🔍 搜索"。点击"✕ 清除"恢复全表显示。

四、数据导出

  • 导出 CSV：点击"📥 导出 CSV"，文件保存到 server/exports/表名.csv，可用 Excel 直接打开。
  • 导出 JSON：点击"📥 导出 JSON"，文件保存到 server/exports/表名.json。

五、撤销与重做

每次增删改操作都会自动记录在历史栈中（最多 50 步）：

  • 撤销 (Undo)：回退到上一步操作前的状态。
  • 重做 (Redo)：恢复被撤销的操作。

撤销后如执行新操作，重做栈将被清空。

六、删除数据表

选中表后，左侧底部出现"🗑 删除当前表"按钮。删除整表会同时删除表中所有数据，但可通过撤销恢复。

💡 提示：项目底层使用 MySQL 8.0 持久化存储，数据安全可靠。如遇连接问题，请检查 MySQL 服务是否启动。
